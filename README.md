# fog-leyu-resource

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面资源。这些页面以静态文件形式存放，不关联任何特定域名或网站，仅作为资源存储与展示用途。

仓库内容可能包括但不限于：工具类页面、信息展示页、临时页面、演示页面等。所有页面均为独立 HTML 文件，便于直接访问或嵌入使用。

## 目录说明

```
fog-leyu-resource/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的公共资源（如 CSS、JS、图片等）
├── README.md       # 本文件
└── .gitignore      # Git 忽略规则
```

- `pages/`：每个 HTML 文件代表一个独立页面，命名清晰，方便识别。
- `assets/`：存放页面所需的公共资源文件，按需组织。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面可能引用外部资源（如字体、图标库），建议使用相对路径或 CDN 链接。
- 归档页面可能不定期更新、替换或移除，恕不另行通知。
- 部分页面可能包含示例数据或占位内容，不代表最终用途。

## 使用方式

可直接通过 GitHub 仓库页面浏览 `pages/` 目录下的 HTML 文件，或使用 Git 克隆到本地后通过浏览器打开对应文件。

```bash
git clone https://github.com/your-username/fog-leyu-resource.git
```

然后打开 `pages/` 中任意 HTML 文件即可查看。
